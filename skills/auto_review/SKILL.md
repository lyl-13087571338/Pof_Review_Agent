This skill parses PDFs, uses a large language model to generate a structured peer review, and outputs a professional Word file.

## Design Thinking
- **Purpose**: Fully automate peer review generation for POF submissions.
- **Tone**: Professional, constructive, clear.
- **Constraints**: Python execution, PDF parsing, LLM API call, Word output.
- **Differentiation**: Combines automation and LLM intelligence into a polished Word report.

## Input / Output
- Input: PDF file path
- Output: Word file path containing structured review
